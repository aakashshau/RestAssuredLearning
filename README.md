# Rest Assured API Testing Project 🚀

This project is a practice framework built using **Java**, **Rest Assured**, **TestNG**, and **Maven** to test APIs provided by [Rahul Shetty Academy](https://rahulshettyacademy.com/). It covers real-world API scenarios like login, adding products, placing orders, and deleting products.

### 🔧 Tools Used

* Java
* Rest Assured
* TestNG
* Maven
* Git & GitHub

### ✅ What This Project Covers

* User login and token handling
* Adding products with image upload
* Placing orders
* Deleting products using path parameters
* OAuth 2.0 authentication
* Sending bug reports to Jira (sample)
* JSON serialization/deserialization using POJOs
* TestNG reports generation

### 📂 Key Files

* `pom.xml` – manages dependencies
* `testng.xml` – test suite setup
* `addPlace.json` – sample payload
* `ECommerceAPITest.java` – main end-to-end test flow
* `ReUsableMethods.java` – common utility methods
* `serializeTest.java` – example for using POJOs

### ▶️ How to Run

1. Make sure Java and Maven are installed
2. Clone the repo:
   `git clone https://github.com/aakashshau/RestAssuredLearning.git`
3. Run the tests:
   `mvn test` or run `testng.xml` from your IDE

📊 Test Reporting

All test results are stored in the test-output folder
Open test-output/index.html in your browser to view the report
